markdown语法

🍭 Markdown语法示例

1.标题[ # ]

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

example:

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

2.强调[ ** 或 __ 包围 ]

**强调字体**
__强调字体__

example:
强调字体
强调字体

3.斜体 [ * 或 _ 包围 ]

*斜体字体*
_斜体字体_

example:
斜体字体
斜体字体

4.删除线 [ ~~包围 ]

~~删除的文字~~

example:
删除的文字

5.高亮 [ == 包围 ]

==高亮字体==

example:
==高亮字体==

6.代码行 [ ` 包围 ]

`print('a')`

example:
printf('a')

7.代码块 [ ‘’’ 包围 ]

example:

8.引用 [ > 并且 > > 前置]

> 一级引用
> > 二级引用

example:

一级引用

二级引用

9.无序列表[ - 或 + 或 * 前置]

- 无序列表
+ 无序列表
* 无序列表

example:

  • 无序列表
  • 无序列表
  • 无序列表

10.有序列表 [ 数字 + . 前置 ]

1. 有序列表
2. 有序列表
3. 有序列表

example:

  1. 有序列表
  2. 有序列表
  3. 有序列表

11. 注释 [ [^] 后置]

这是注释<sup id="fnref:1" class="footnote-ref"><a href="#fn:1" rel="footnote"><span class="hint--top hint--rounded" aria-label="注释的解释">[1]</span></a></sup>

example:
这是注释[1]

12. 链接 [常用 “[ ]” + “( )” 分别包围文本与链接]

[正文](链接)

13.任务列表 [“- [ ]” + 空格 前置]

"- [ ] 刷B站"
"- [ ] 写代码"
"- [x] 起床"

example:

  • 刷B站
  • 写代码
  • 起床

14.表格 [用 “|” 绘制表格边框]

| 学号 | 姓名  | 年龄 |
| :--- | :---: | ---: | (引号的位置代表着 左对齐, 居中, 右对齐)
|1|a|24|
|2|b|25|

example:

学号 姓名 年龄
1 b 24
2 b 25

15.图片 [ “! [图片] (图片地址)” ]

{% asset_img example.png This is an test image %}

***
---
___
